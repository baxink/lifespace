# 家庭物品清单 - 项目规格说明

## 1. 项目概述

- **项目名称**: 家庭物品清单 (Lifespace)
- **项目类型**: 响应式Web应用（静态页面）
- **核心功能**: 通过扫描二维码快速查找和记录家庭物品的存放位置
- **目标用户**: 家庭成员，尤其是老年人

## 2. 技术方案

### 技术栈
- **前端**: 纯HTML + CSS + JavaScript（无框架）
- **QR扫描**: html5-qrcode 库
- **数据存储**: LocalStorage（浏览器本地存储）
- **部署方式**: 阿里云OSS 或 GitHub Pages

### 文件结构
```
Lifespace/
├── index.html      # 主页 - 物品列表
├── add.html        # 添加页 - QR扫码录入
├── item.html      # 详情页 - 查看物品信息
├── edit.html      # 编辑页 - 修改物品信息
├── css/
│   └── style.css  # 样式表
├── js/
│   ├── storage.js # 数据存储模块
│   ├── scanner.js # QR扫描模块
│   └── app.js     # 主应用逻辑
└── SPEC.md        # 本规格文档
```

## 4. 数据模型

```javascript
{
  id: String,          // QR码扫描内容，作为唯一标识
  name: String,        // 物品名称
  location: String,    // 放置位置
  category: String,    // 分类 (electronics/documents/tools/others)
  image: String,       // 图片Base64编码
  createdAt: Number,   // 创建时间戳
  updatedAt: Number    // 更新时间戳
}
```

## 6. 部署指南

### 阿里云OSS
1. 将所有文件打包
2. 创建OSS Bucket，设置为公共读
3. 上传所有文件
4. 访问Bucket域名即可

### GitHub Pages
1. 创建GitHub仓库
2. 推送代码
3. Settings → Pages → 选择分支
4. 等待部署完成

## 7. 使用流程

1. **准备阶段**：在物品上贴好QR码（二维码内容为物品的唯一编号）
2. **首次使用**：打开网页，点击"+"按钮，扫描物品上的QR码
3. **录入信息**：填写物品名称、位置，拍照或上传图片
4. **查找物品**：打开主页，通过搜索或浏览找到物品
5. **查看位置**：点击物品卡片，查看详细的存放位置
